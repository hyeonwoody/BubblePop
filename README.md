## 🧑‍💻: Intro
브이로그 영상 촬영 및 편집은 시간이 많이 소비되는 작업입니다.
영상을 실시간으로 시청하면서 
코멘터리 형식으로 영상 속 적재적소의 타이밍에
내가 하고 싶은 말을 전달하는 자막을 작업할 때,
편하게 타이핑 만으로 자막편집을 할 수 있는
웹 기반 말풍선 생성기를 만들어보았습니다.
